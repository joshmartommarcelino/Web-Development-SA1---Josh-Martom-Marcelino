HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. Here's the basic structure of an HTML document:

Basic HTML Structure
<!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
  </html>

Text Tags

<h1> through <h6> - Headings

HTML provides six levels of headings, from <h1> (most important) to <h6> (least important).

<h1>Main Heading</h1>
  <h2>Subheading</h2>
  <h3>Section Heading</h3>
  <h4>Subsection Heading</h4>
  <h5>Small Heading</h5>
  <h6>Smallest Heading</h6>

<p> - Paragraph

The paragraph tag defines a paragraph of text.

<p>This is a paragraph of text. Paragraphs are block-level elements.</p>

<a> - Anchor (Links)

The anchor tag creates hyperlinks to other web pages or resources.

<a href="https://www.example.com">Visit Example Website</a>
  <a href="page2.html">Go to Page 2</a>
  <a href="#section-id">Jump to Section</a>

<strong> and <em> - Text Emphasis

These tags are used to emphasize text. <strong> indicates text of strong importance (typically bold), while <em> indicates emphasized text (typically italic).

<p>This is <strong>important</strong> information.</p>
  <p>This is <em>emphasized</em> text.</p>

<code> - Code Text

The code tag is used to display programming code or monospaced text.

<p>The HTML tag for a paragraph is <code>&lt;p&gt;</code>.</p>

<pre> - Preformatted Text

The pre tag preserves both spaces and line breaks, useful for displaying code blocks.

<pre>
  function sayHello() {
    console.log("Hello, world!");
  }
  </pre>

Structural Tags

<div> - Division

A generic container used to group content for styling and layout purposes.

<div class="container">
    <h2>Section Title</h2>
    <p>Content goes here...</p>
  </div>

<span> - Inline Container

An inline container for grouping text or elements within a line.

<p>This paragraph contains <span class="highlight">highlighted text</span> within it.</p>

<header>, <nav>, <main>, <section>, <article>, <footer>

Semantic structural elements that describe the purpose of different parts of a webpage.

<header>
    <h1>Website Title</h1>
  </header>
  
  <nav>
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
  </nav>
  
  <main>
    <section>
      <h2>Section Title</h2>
      <article>
        <h3>Article Title</h3>
        <p>Article content...</p>
      </article>
    </section>
  </main>
  
  <footer>
    <p>© 2025 Example Company</p>
  </footer>

Media Tags

<img> - Image

Embeds an image in the HTML document.

<img src="image.jpg" alt="Description of image" width="300" height="200">

<video> - Video

Embeds a video player in the document.

<video controls width="320" height="240">
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.webm" type="video/webm">
    Your browser does not support the video tag.
  </video>

<audio> - Audio

Embeds an audio player in the document.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser does not support the audio element.
  </audio>

Form Tags

<form> - Form Container

The form tag is used to create an HTML form for user input.

<form action="/submit" method="post">
      <!-- Form elements go here -->
    </form>

<input> - Input Field

The input tag is used to create various input controls.

<!-- Text input -->
    <input type="text" name="username" placeholder="Enter username">
    
    <!-- Password input -->
    <input type="password" name="password" placeholder="Enter password">
    
    <!-- Email input -->
    <input type="email" name="email" placeholder="Enter email">
    
    <!-- Number input -->
    <input type="number" name="age" min="18" max="100">
    
    <!-- Checkbox -->
    <input type="checkbox" name="subscribe" id="subscribe">
    <label for="subscribe">Subscribe to newsletter</label>
    
    <!-- Radio buttons -->
    <input type="radio" name="gender" id="male" value="male">
    <label for="male">Male</label>
    <input type="radio" name="gender" id="female" value="female">
    <label for="female">Female</label>
    
    <!-- Date input -->
    <input type="date" name="birthday">
    
    <!-- File input -->
    <input type="file" name="document">
    
    <!-- Hidden input -->
    <input type="hidden" name="user_id" value="123">
    
    <!-- Submit button -->
    <input type="submit" value="Submit">

<label> - Input Label

The label tag defines a label for an input element.

<label for="username">Username:</label>
    <input type="text" id="username" name="username">

<select> and <option> - Dropdown List

The select tag creates a dropdown list, and option tags define the available options.

<label for="country">Country:</label>
    <select id="country" name="country">
      <option value="">Select a country</option>
      <option value="us">United States</option>
      <option value="ca">Canada</option>
      <option value="uk">United Kingdom</option>
    </select>

<textarea> - Multiline Text Input

The textarea tag creates a multiline text input field.

<label for="message">Message:</label>
    <textarea id="message" name="message" rows="4" cols="50">
      Default text here...
    </textarea>

<button> - Clickable Button

The button tag creates a clickable button.

<button type="submit">Submit Form</button>
    <button type="reset">Reset Form</button>
    <button type="button" onclick="alert('Hello World!')">Click Me</button>

<fieldset> and <legend> - Form Sections

The fieldset tag groups related form elements, and the legend tag provides a caption for the fieldset.

<fieldset>
      <legend>Personal Information</legend>
      <label for="fname">First name:</label>
      <input type="text" id="fname" name="fname">
      <label for="lname">Last name:</label>
      <input type="text" id="lname" name="lname">
    </fieldset>

CSS Basics

CSS Syntax

CSS (Cascading Style Sheets) is used to style HTML elements. The basic syntax consists of a selector and a declaration block.

selector {
      property: value;
      property: value;
    }

CSS Selectors

CSS selectors are used to target specific HTML elements for styling.

/* Element selector */
    p {
      color: blue;
    }
    
    /* Class selector */
    .highlight {
      background-color: yellow;
    }
    
    /* ID selector */
