HTML & CSS Quick-Reference Guide

HTML Tags

  • <h1> to <h6>: Headings (h1 is the largest, h6 is the smallest).
  • <p>: Paragraph for text content.
  • <a href="url">: Anchor tag for hyperlinks.
  • <ul> and <ol>: Unordered and ordered lists.
  • <li>: List item for use inside <ul> or <ol>.
  • <img src="image.jpg" alt="description">: Image tag.
  • <div>: Division or section in a document.
  • <span>: Inline container for styling.
  • <table>, <tr>, <td>: Table, table row, and table data.
  • <form>, <input>: Form and input elements.

CSS Properties

        
body {
    font-family: Arial, sans-serif; /* Sets the font */
    color: #333; /* Text color */
    background-color: #f9f9f9; /* Background color */
    margin: 0; /* Removes default margin */
    padding: 20px; /* Adds padding */
}

h1 {
    text-align: center; /* Centers text */
    color: #444; /* Heading color */
}

ul {
    list-style-type: square; /* Changes bullet style */
    padding: 15px; /* Adds padding */
}

a {
    text-decoration: none; /* Removes underline from links */
    color: #007BFF; /* Link color */
}

img {
    border-radius: 5px; /* Rounds image corners */
    box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1); /* Adds shadow */
}
